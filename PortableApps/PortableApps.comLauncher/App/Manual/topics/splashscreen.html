Splash screensÂ¶

PortableApps.com applications usually have a splash screen. This file goes in App\AppInfo\splash.jpg and must be a JPEG image. The usual procedure is to use the black and white PortableApps.com Development Test Release splash screen while an application is a Development Test and then when it is deemed ready to become a Pre-Release, it will be given its own one ready for becoming an official PortableApps.com release.

Configuration values which affect splash screens are [Launch]:SplashTime and [Launch]:LaunchAppAfterSplash.
