Features for the user
=====================

This is not a comprehensive list at all and has yet to be filled out a lot.

* Data restoration in case of disk removal or power failure.

* "Live mode" to run in a read-only way, such as from a CD.

* Easy integration with the PortableApps.com Platform.
  
* Automatic language switching in most apps based on the language in the
  PortableApps.com Platform.

* Easy backup with the backup utility in the PortableApps.com Platform.
