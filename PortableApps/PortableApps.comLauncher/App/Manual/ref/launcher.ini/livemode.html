[LiveMode]¶

The [LiveMode] section provides details regarding how the portable application should function when in Live mode, when in a read-only directory.

As specified here, certain things are copied to the local hard drive, and then the application is run, and any things copied locally are deleted. This is called running locally, or “Live mode”.

CopyApp¶

Values: true / false
Default: true
Optional.

When run in Live mode, does the application need to be run from a writable location? If yes, specify true.

CopyData¶

Obsolete.

This option has been removed due to issues with the %PAL:DataDir% replacement variable in file moving. Data will now always be copied. This may change back in the future.
