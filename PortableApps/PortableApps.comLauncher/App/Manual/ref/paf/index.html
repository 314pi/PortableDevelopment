PortableApps.com Format™ Specification¶

2.0 (2011-04-12)

PortableApps.com Format is a simple specification that governs the file and directory layout as well as operating behavior of portable apps. File are distributed as easy-to-use .paf.exe™ installer files. The specification is broken into sections for easy reading.

  • 1. Directory and File Layout
  • 2. AppInfo.ini (App Configuration)
  • 3. Icons
  • 4. PortableApps.com AppCompactor and AppCompactor.ini
  • 5. PortableApps.com Installer (and installer.ini)
  • 6. Host PC Modifications & Portability
  • 7. Plugin Installers
  • 8. PortableApps.com Format Version History and Discussion
