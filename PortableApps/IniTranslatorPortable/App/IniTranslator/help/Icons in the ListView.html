Icons in the ListView

The list view displays icons to the left of each item. You can use these icons to quickly determine the status of an item. The list view can display any of the following four icons:

 

 - the item is translated and there is either a comment in the original or the translated file or both

 - the item is not translated and there is either a comment in the original or the translated file or both

 - the item is translated and there are no comments

 - the item is not translated and there are no comments

 

To view available comments, press Ctrl+M when the item is selected.

 

Additionally, a small marker is shown to the left of the icon if the item has been modified. Note that this marker is not visible if you set a bookmark on the item since the bookmark and the marker share the same space.
